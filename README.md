# AuraMed - Clínica Virtual

AuraMed es una aplicación web Frontend de Telemedicina y Triaje Virtual. Está diseñada con una estética premium, minimalista y modo oscuro inspirado en Apple.

## Funcionalidades
- **Triaje Virtual Simulado**: Chat interactivo para describir síntomas con respuestas automatizadas en JavaScript.
- **Detección de Emergencias**: Si el usuario escribe palabras clave críticas (como "emergencia", "arritmia", "sangrado", etc.), el sistema lanza una alerta médica de prioridad alta.
- **Especialidades Médicas**: Tarjetas interactivas para simular la derivación a distintas unidades (Medicina General, Cardiología, Pediatría).
- **Pop-up de Citas**: Flujo sin recargar página para registrar citas presenciales de urgencia con mensajes de éxito integrados.

## Tecnologías Utilizadas
- HTML5
- CSS3 (Variables, Flexbox, CSS Grid)
- JavaScript Vanilla (Sin frameworks ni librerías de terceros)

## Instalación y Uso
1. Clona el repositorio:
   ```bash
   git clone https://github.com/Stephanoescu/AuraMed.git
   ```
2. Abre el archivo `index.html` en tu navegador preferido. ¡No requiere servidor ni instalación de paquetes!
